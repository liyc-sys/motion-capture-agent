========================================
  网页素材采集器 · 使用说明
========================================

【怎么干活】
1. 打开程序:
   - Mac:双击「start.command」(首次右键→打开→打开)
   - Windows:双击「MotionCapture.exe」
     (首次弹"Windows 保护了你的电脑"→ 点「更多信息」→「仍要运行」)
2. 弹出窗口,提示你输入网站地址
3. 输入要采集的网站地址(例如 www.example.com),按回车
4. 浏览器会打开该网站,你正常浏览:
   - 上下滚动看看
   - 点开里面的链接看(点链接跳到的新页面,也会一起采集进来)
   - 看图片、看视频都行
5. 看完了,直接关掉浏览器窗口,这条就完成
   (关掉后程序会自动保存剩余素材,几秒钟,不用操作)
6. 想做下一个站:再双击一次程序,输入下一个网址
   —— 每次打开只录一个站,关掉浏览器 = 这条做完

【素材存哪】
- 都存在你桌面的「motion_capture_<你的电脑名>」文件夹里
- 每个站单独一个子文件夹(按顺序编号),不会混

【做完怎么交】
- 把桌面那个「motion_capture_...」文件夹整个压缩成 zip,发回给我
- 做多少算多少,每条独立

【注意】
- 需要电脑装了 Chrome 或 Edge(绝大多数电脑都有,没有就告诉我)
- 浏览时尽量多滚动、多点开链接看看,看得越细采集越全
- 只存图片/视频/音效,不会存你的任何隐私信息
- 有打不开、报错的情况,截图发我
