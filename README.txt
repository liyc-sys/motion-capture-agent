========================================
  网页素材采集器 · 使用说明
========================================

【第一次打开(重要!照做就不会被 Mac 拦)】
1. 解压后,进入 MotionCapture 文件夹
2. 找到「MotionCapture.exe」,双击打开;若弹「Windows 保护了你的电脑」→ 点「更多信息」→「仍要运行」
3. 万一右键打开还是被拦(极少数),用终端兜底:
   - 打开「终端」(在启动台搜索"终端",打开它)
   - 输入这行字(注意有个空格): xattr -dr com.apple.quarantine
   - 然后把 MotionCapture 文件夹整个拖到终端窗口里,松手,回车
   - 没有任何提示就是成功了,再去双击 MotionCapture.exe

【怎么干活】
1. 第一次右键 MotionCapture.exe → 打开;以后双击就行
2. 弹出窗口提示输入网站地址,输入(例如 www.example.com),回车
3. 浏览器打开该网站,正常浏览:上下滚动、点开链接看
   (点链接跳到的新页面,也会一起采集进来)
4. 看完直接关掉浏览器窗口
5. ⚠️ 关掉浏览器后,黑窗口会开始下载素材并显示进度
   【一定要等它提示"✅ 已写入完成标志 DONE.txt"再关窗口!】
   提前关窗口 = 这条素材不完整,要重做
6. 想做下一个站:再双击 MotionCapture.exe,输入下一个网址

【素材存哪】
- 桌面「motion_capture_<你的电脑名>」文件夹,每个站一个子文件夹
- 每条做完的文件夹里会有一个 DONE.txt(完成标志);没有它就是没做完

【做完怎么交】
- 把桌面那个文件夹整个压缩成 zip 发回

【注意】
- 需要电脑装了 Chrome 或 Edge
- 多滚动、多点开链接,看得越细采集越全
- 只存图片/视频/音效,不存隐私
- 报错截图发回
