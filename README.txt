========================================
  网页素材采集器 · 使用说明
========================================

【第一次打开】
1. 双击「启动.command」
2. Mac 可能弹「无法打开,因为来自身份不明的开发者」
   → 不要怕,这样处理:右键点「启动.command」→ 选「打开」→ 再点「打开」
   (只第一次需要,以后双击就行)
3. 会弹出一个小黑窗(终端)和一个 Chrome 浏览器窗口

【怎么干活】
- 浏览器会自动打开第一个要看的网站
- 你就正常浏览这个网站:上下滚动、点开里面的链接、看图片看视频
- 你看到的所有图片/视频/音效,会自动偷偷存到你电脑里
- 看完一个站,点屏幕右下角蓝色「下一个 ▶」按钮,看下一个站
- 右下角会显示:第几个站 / 已采集多少个素材

【素材存在哪】
- 都存在你桌面的「motion_capture_<你的电脑名>」文件夹里
- 小黑窗启动时也会打印这个路径

【做完怎么交】
- 把桌面那个「motion_capture_...」文件夹整个压缩成 zip,发回来就行
- 不用关浏览器,随时可以关;下次打开会接着没做完的站继续

【常见问题】
- 浏览器没弹出来?看小黑窗有没有报错,截图发回
- 提示找不到浏览器?本程序需要电脑装了 Chrome 或 Edge(一般都有)
- 想换一批网站看?把同目录的「queue.txt」换成新清单(一行一个网址)即可
